## About Application

* Interactive 
* Map Based Solution
* User Friendly
* Citizen Oriented 
* Officer Oriented 
* Real Time Processing 


## Features

* User can report the crime.
* Know about crime prone zones.
* Generate an SOS alarm in emergency.
* Map assistance for more awareness.
* Establish network for effective communication and coordination amongst citizens and police.
* Location Sharing and Tracking.
* Increase the coverage of area being monitored.
* Light application, suits all models and system versions.
* Geo-Fencing, Family Meter, Simple UI, Native Language Support make it a one-stop application; suitable for all.
* ML Assisted backend for precise geo-fence and limiting false calls and alarms ensures accuracy.
